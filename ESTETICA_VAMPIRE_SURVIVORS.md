# Estética Vampire Survivors - Pixel Art ✅

## Mudanças Implementadas

### 1. Personagem Herói (Player) ✅
O SpriteGenerator já possui um herói pixel art completo:
- Guerreiro com armadura azul ciano
- Capacete com detalhes dourados
- Capa roxa flutuante
- Design 16x16 escalado para 48x48 pixels
- Vista top-down estilo Vampire Survivors

### 2. Tela de Upgrade Renovada ✅
Transformada para estilo Vampire Survivors:

**Características:**
- Cards grandes e visuais (260x360px)
- Ícones emoji grandes (72px)
- Sistema de raridade com cores:
  - Comum: Cinza escuro
  - Raro: Azul
  - Épico: Roxo
- Bordas pixel art (sem cantos arredondados)
- Animação de entrada dos cards (fade + scale)
- Animação de saída suave
- Fundo escuro semi-transparente

**Upgrades com Nomes Temáticos:**
- ⚔ ESPADA AFIADA (Dano)
- 👟 BOTAS RÁPIDAS (Velocidade)
- 🎯 OLHO DE ÁGUIA (Alcance)
- ⚡ FÚRIA VELOZ (Cooldown)
- ❤ CORAÇÃO FORTE (HP)
- 🔱 TIRO MÚLTIPLO (Multishot)
- 🎲 SORTE FATAL (Crítico)
- 💥 GOLPE MORTAL (Dano Crítico)
- 🩸 VAMPIRISMO (Lifesteal)
- 👻 PASSO SOMBRIO (Dash)

### 3. HUD Melhorada ✅
Interface estilo Vampire Survivors:

**Superior Esquerdo:**
- Barra de HP vermelha com borda pixel art
- Barra de XP verde com animação de preenchimento
- Label de nível com estrela ⭐
- Animação de pulso quando HP baixo
- Animação de level up (escala)

**Superior Direito:**
- Tempo de sobrevivência ⏱
- Sistema de combo 🔥 com cores dinâmicas:
  - 1-10: Amarelo
  - 11-20: Laranja
  - 21+: Vermelho + escala maior
- Stats compactas (dano, crítico, lifesteal, projéteis, cooldown)

**Estilo Visual:**
- Bordas pixel art (3px)
- Sem cantos arredondados
- Cores escuras de fundo
- Texto grande e legível

### 4. Game Over Screen ✅
Tela de fim de jogo renovada:

**Características:**
- Fundo escuro (90% opacidade)
- Painel central com borda vermelha (6px)
- Título "💀 GAME OVER 💀" em vermelho
- Tempo sobrevivido em amarelo dourado
- Estatísticas finais da run:
  - Nível alcançado
  - Dano final
  - Número de projéteis
  - Chance de crítico
  - Lifesteal (se houver)
- Botão grande "🔄 JOGAR NOVAMENTE"
- Animação de fade in

### 5. Sprites Pixel Art ✅
O SpriteGenerator já possui:

**Herói:**
- Guerreiro 16x16 (48x48 escalado)
- Armadura detalhada
- Capa e capacete
- Animação de bobbing

**Inimigos:**
- Demônio/zumbi vermelho
- Inimigo rápido (ciano)
- Inimigo tanque (laranja)
- Boss épico (roxo com aura)

**Projéteis:**
- Orbe mágico dourado
- Laser azul elétrico
- Bomba explosiva

**Efeitos:**
- Partículas de XP verdes
- Números de dano flutuantes
- Efeito de level up (círculo dourado)

## Como Testar

1. Abra o projeto no Godot
2. Execute a cena Main (F5)
3. Observe:
   - Herói pixel art (não mais nave)
   - HUD com barras pixel art
   - Mate inimigos para ganhar XP
   - Suba de nível para ver os cards de upgrade
   - Morra para ver a tela de game over
